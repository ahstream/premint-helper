Shortcut links

Shortcut links below can be used to run Premint Helper functions from other apps, or save them as bookmarks and run from browser bookmarks bar.
  • any-link
  • reveal-alphabot-raffles
  • show-alphabot-results
  • update-alphabot-results
  • close-tabs
  • close-window
  • close-tabs-minimize-window
  • minimize-window
Example usage:
  1. The any-link link will open a new browser tab and load supplied URL. If link is a Alphabot or Premint raffle, Premint Helper will automatically Auto Join that raffle. An external application could use this mechanic to automatically join raffles in multiple browser instances.
  2. The update-alphabot-results link could be used in a similar way to update results for multiple browser instances in one go.
  3. The close-* and minimize-* link can be used to simplify things if using multiple browser instances. For example, close-tabs-minimize-window could be used to close all tabs when finished in one browser window and quickly move on to next window.

Storage

Chrome extensions are limited to storing 5 MB of data, which eventually will not be enough to store raffle results and other data. Premint Helper therefore uses the unlimitedStorage permission.
