Bookmark commands

Bookmarks below can be used to run Premint Helper functions.
  • any-link
  • reveal-alphabot-raffles
  • show-alphabot-results
  • update-alphabot-results
  • close-tabs
  • close-window
  • close-tabs-minimize-window
  • minimize-window
Example usages of bookmark commands:
  1. The any-link bookmark will open a new browser tab and load supplied URL. If link is a Alphabot or Premint raffle, Premint Helper will automatically Auto Join that raffle. An external application could use this mechanic to automatically join raffles in multiple browser instances.
  2. The update-alphabot-results bookmark could be used in a similar way to update results for multiple browser instances in one go.
  3. The close-* and minimize-* bookmarks can be used to simplify things if using multiple browser instances. For example, close-tabs-minimize-window could be used to close all tabs when finished in one browser window and quickly move on to next window.

Storage

Chrome extensions are limited to storing 5 MB of data, which eventually will not be enough to store raffle results and other data. Premint Helper therefore need to use the unlimitedStorage permission. If storage is getting too big for your liking, you can remove and re-install the browser extension to start over.
